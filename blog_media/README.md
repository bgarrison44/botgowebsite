# Blog Media Folder

Store self-hosted blog assets here.

Examples:
- blog_media/week-01-update.mp4
- blog_media/voice-note-2026-02-17.m4a
- blog_media/dashboard-screenshot.png

Then reference these files in `blog-content.js`, for example:
- `mediaUrl: "blog_media/week-01-update.mp4"`
- `images: [{ src: "blog_media/dashboard-screenshot.png", alt: "Dashboard screenshot" }]`
